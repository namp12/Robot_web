# 05 - Frontend Specification

Frontend xây dựng theo **Component-Based Architecture** phong cách Dashboard hiện đại (Slate Dark Theme).

- **Công nghệ:** React 19, TypeScript, Vite, React Router v6, Tailwind CSS v3, Axios, WebSocket Client.
- **11 Trang Giao diện:** Dashboard, Robot Control, Camera, LiDAR, SLAM, Navigation, Maps, AI Assistant, Black Box, System Monitor, Settings.
