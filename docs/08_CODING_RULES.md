# 08 - Coding Rules & Style Guidelines

- **Python:** PEP8 compliant, Type Hints đầy đủ, Async/Await cho I/O, sử dụng Pydantic v2 & SQLAlchemy 2.x Mapped models.
- **TypeScript/React:** Strict TypeScript, Functional Components với React Hooks, Tailwind utility classes.
- **Clean Architecture:** Router ➔ Service ➔ Database / ROS2 Bridge. Không hardcode hay dùng global variables bừa bãi.
